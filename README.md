# 기본 사용법:
  - ```/돈``` 이 기본 명령어로 뒤에 옵션이 3가지가 있다
  - ```/돈 확인 [-닉네임]``` (- 는 써도 되고 안써도 된다는 뜻)
  - ```/돈 설정 [닉네임] [정수]``` (돈 범위 : –2,147,483,648 ~ 2,147,483,647 (int 자료형입니다~))
  - ```/돈 송금 [닉네임] [정수]``` (돈 범위 : –2,147,483,648 ~ 2,147,483,647 (int 자료형입니다~))
